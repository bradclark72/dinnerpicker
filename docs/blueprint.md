## Core Features:

- Location Detection: Automatically detect the user's current GPS location using the Google API: AIzaSyCkGePqEnaQBUj5g2Ia6_vxKQiWRrboJrQ.
- Cuisine Selection: Allow users to select their preferred cuisine(s) from a predefined list (e.g., Italian, Mexican, Chinese).
- Radius Filter: Allow users to specify a maximum distance (mileage) they are willing to travel.
- Restaurant Search: Use the Google API to search for restaurants matching the selected cuisine(s) within the specified radius of the user's location.
- AI Restaurant Ranker: An AI tool that uses available data and user preference factors to decide which matching restaurant to display. Considerations may include reviews, price and popularity.
- Restaurant Display: Display the chosen restaurant's name, address, phone number, and user ratings, fetched through the api.

## Style Guidelines:

- Primary color: Warm gold (#E4A223) to evoke a sense of culinary richness and warmth.
- Background color: Light beige (#F5F5DC), offering a neutral and inviting backdrop.
- Accent color: Muted brown (#A67B5B), providing earthy tones for contrast and sophistication.
- Font: 'PT Sans', a modern yet accessible sans-serif font suitable for both headings and body text.
- Use simple, outlined icons representing different cuisines and filters.
- Clean and intuitive layout with a focus on easy filtering and restaurant information display.
- Subtle transition animations when searching and displaying restaurant results.
